# Task: 项目初始化

## 描述

搭建 Merx CLI 项目的基础结构，包括 package.json、TypeScript 配置、unbuild 构建配置和基本目录结构。

## 验收标准

- [ ] package.json 包含正确的 name、bin、type: module 配置
- [ ] tsconfig.json 配置为 ESM + Node 18+ 目标
- [ ] unbuild 配置 (build.config.ts) 正确设置
- [ ] 基本目录结构创建完成 (src/、src/commands/、src/utils/)
- [ ] 依赖安装成功：beautiful-mermaid、citty
- [ ] `npm run build` 可以成功执行

## 技术细节

### package.json 关键配置
```json
{
  "name": "merx",
  "type": "module",
  "bin": {
    "merx": "./dist/cli.mjs"
  },
  "exports": {
    ".": {
      "import": "./dist/index.mjs"
    }
  }
}
```

### tsconfig.json 要点
- target: ES2022
- module: ESNext
- moduleResolution: bundler
- strict: true

### 目录结构
```
src/
├── cli.ts
├── commands/
├── utils/
└── config.ts
```
